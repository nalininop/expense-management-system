# Expense-management-system

A modern Expense Tracker Dashboard built using HTML, CSS, and JavaScript. This project helps users manage their budget, track daily expenses, and monitor spending through an interactive dashboard.

## 📌 About the Project

I developed this project to strengthen my frontend development skills and gain practical experience with JavaScript DOM manipulation, Local Storage, and data visualizatio
Live Demo - expensemanagement-system.netlify.app
The application allows users to:

* Set and update their budget
* Add expense transactions
* Categorize expenses using tags
* Edit and delete transactions
* View total budget, total expenses, and remaining balance
* Sort transaction history
* Store data in browser Local Storage
* Visualize spending using Chart.js

## 🔧 How to Run

1. Download or clone the repository.
2. Open the project in VS Code.
3. Install the Live Server extension.
4. Right-click on `index.html`.
5. Click **Open with Live Server**.
